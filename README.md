# [Purkynthon](https://purkynthon.online) - Purkiáda 2026    
Purkynthon is an educational web-based game for 8./9. graders. Created for the event `Purkiáda 2026`. It is based around connecting random nodes with random code to complete a set of tasks that your employer gives you.

---

## Features

- **Visual node editor**: An interface for building logic visually

- **Code execution**: Node graphs are converted into executable code and run instantly (with funny small execution texts)

- **(Retarded) AI assistant**: An AI helper that gives hints, explanations, and unhelpful commentary

---

## Architecture

### Frontend

- **React** – all of frontend
- **Tailwind CSS** – epic styling
- **React Flow** – node editor
- **CodeMirror** – code viewing and editing

### Backend

- **Python** – core backend magic
- **Ollama** – local LLM powering the very useful and smart AI assistant
