# Purkynthon - Purkiáda 2026    
Purkynthon is a educational web-based game for 8./9. grades for the event `Purkiáda 2026`.
Based around connecting random nodes with random code to complete a set of tasks that your employer gives you.

---

## Features

- **Visual node editor**: A interface for building logic visually, converting programing to a half visual expirience

- **Real-time code execution**: Node graphs are converted into executable code and run instantly (with funny small execution texts)

- **(Retarded) AI assistant**: An AI helper that gives hints, explanations, and unhelpful commentary

---

## Architecture

### Frontend

- **React** – all of frontend
- **Tailwind CSS** – epic styling
- **React Flow** – node editor
- **CodeMirror** – code viewing and editing

### Backend

- **Python** – core backend magic
- **Ollama** – local LLM powering the very useful and smart AI assistant
